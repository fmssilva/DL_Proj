        "---\n",
        "## Task 1 — MLP Baseline\n",
        "\n",
        "A fully-connected Multi-Layer Perceptron treating each 64×64 RGB image as a **flat vector of 12 288 features**.\n",
        "\n",
        "**Architecture:** `Flatten → FC(12288→512) → BN → ReLU → Drop(0.4) → FC(512→256) → BN → ReLU → Drop(0.4) → FC(256→128) → BN → ReLU → Drop(0.4) → FC(128→9)`  \n",
        "Total parameters: ~6.4 M.\n",
        "\n",
        "| Design choice | Decision | Rationale |\n",
        "|---|---|---|\n",
        "| Input | Flatten 64×64×3 → 12 288 | MLP required — no convolutions |\n",
        "| Hidden layers | 3 × (FC → BN → ReLU → Dropout) | depth adds capacity; 3 is enough before diminishing returns |\n",
        "| Width schedule | 512 → 256 → 128 | funnel forces compression / feature abstraction |\n",
        "| Batch Norm | after every FC | stabilises gradients with large flat input; allows higher LR |\n",
        "| Dropout | p=0.4 | regularisation for ~6 M params on a ~1 400-image train set |\n",
        "| Output | FC(128 → 9), no softmax | `CrossEntropyLoss` applies log-softmax internally |\n",
        "| Loss | CrossEntropyLoss(weight=class_weights) | inverse-frequency weights correct 2.76× class imbalance |\n",
        "| Optimiser | Adam lr=1e-3 | adaptive LR; outperforms SGD on small datasets |\n",
        "| Scheduler | StepLR(step_size=5, γ=0.5) | halves LR every 5 epochs — prevents late-stage oscillation |\n",
        "| Early stopping | patience=5 on val_loss | saves best checkpoint, avoids over-fitting |\n",
        "| Image size | 64×64 | balances detail vs. MLP parameter count |\n",
        "| Normalisation | ImageNet mean/std | EDA shows dataset stats close to ImageNet — safe default |\n",
        "| Metric | macro-F1 (not accuracy) | accuracy inflated on imbalanced data; macro-F1 penalises minority-class failures equally |\n",
        "\n",
        "**Tip:** `FAST_RUN = True` at the top of Cell 1 runs the full pipeline on ~54 images in seconds. Flip to `False` for the real 30-epoch Colab run.\n"
